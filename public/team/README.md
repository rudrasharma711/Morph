# Team photos

Drop the team headshots here, then set the `photo` field on the matching member
in `app/team/page.tsx` (e.g. `photo: "/team/rudra.jpg"`).

Suggested files (square images, ~600×600+, JPG/PNG/WebP):
- rudra.jpg     → Rudra Sharma (Founder & CEO)
- kevin.jpg     → Kevin Davies (CFO)
- kasinadh.jpg  → Kasinadh Sudeep (COO)

Until a photo is set, the card shows a monochrome initials placeholder.
